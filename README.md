### Exercício 01

Reprodução de layout com quatro seções empilhadas verticalmente:

* Duas seções com botões lado a lado (mesma largura)
* Um card maior com conteúdo interno
* Dois blocos lado a lado
* Um card menor na parte inferior
* Todas as seções possuem a mesma largura e respeitam as margens da tela

### Exercício 03

Criação de componente reutilizável:

* 3 cards empilhados verticalmente
* Cada card contém:

  * Parte superior com avatar + textos (título e subtítulo)
  * Parte inferior com botão ocupando toda a largura
* O card foi implementado como componente reutilizável

### Exercício 04

Layout responsivo com proporções de altura:

* 4 seções verticais com alturas proporcionais
* Cabeçalho fino no topo
* Área principal maior (≈ 4x o cabeçalho)
* Linha com dois cards lado a lado
* Rodapé com mesma altura do cabeçalho
* Nenhuma altura definida em pixels (uso de flex)

### Como executar o projeto
1. Clone o repositório:
git clone `https://github.com/CaarlosRiian/Flex-App-DDM.git`
3. Acesse a pasta
cd `flex-app`
5. Instale as dependências
`npm install`
7. Execute o projeto
`npx expo start`
